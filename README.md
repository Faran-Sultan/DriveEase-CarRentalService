# DriveEase-CarRentalService

## Project Summary
DriveEase Car-Rental-Service is a responsive website developed using **HTML5 and CSS3** as part of the **AICT Lab Semester Project**.  
The website provides an intuitive car rental experience, presenting services, vehicle listings, and contact information through a clean and user-friendly interface.

This project follows a complete academic workflow including website development, documentation, data analysis, presentations, GitHub versioning, and publishing.
